Object

すべてのオブジェクトの祖先に当たるオブジェクトです。評価すると、自身を返します。

ペアレント

Object にペアレントはありません。

メソッド

eval()
self を評価して返します。
child()
self のチャイルドを返します。
(==)(obj)
self と obj が等価ならば true、それ以外は false を返します。
(!@)()
self が偽ならば true、それ以外は false を返します。
is(obj)
self と obj が同一ならば true、それ以外は false を返します。
to_s()
self を文字列化したものを返します。
slots()
self の全スロットをハッシュにして返します。 ハッシュに対する操作は self には反映されません。

その他のメソッドの定義は、初期化ファイル群の中の object.cy にあります。
