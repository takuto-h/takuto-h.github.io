indexer

dict["ant"] のように記述する、オブジェクトにアクセスするためのオブジェクトです。 評価すると、target.([])(index) と記述した時と同じ結果を返します。

'dict["ant"] のように記述すると、スロット target に dict が、 スロット index に "ant" がセットされた indexer を得ることができます。

ペアレント

Indexer

スロット

target
アクセスの対象となるオブジェクトです。
index
アクセスにおいて場所を示すオブジェクトです。

メソッド

(=)(value)
target.([]=)(index, value) と記述した時と同じ結果を返します。

その他のメソッドの定義は、初期化ファイル群の中の methods.cy にあります。
