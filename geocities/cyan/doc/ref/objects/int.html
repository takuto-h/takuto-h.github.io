数値

123 や 456 など、数値を表すオブジェクトです。評価すると、自身を返します。

ペアレント

Int

メソッド

(<)(n)
self が n より小さければ true、それ以外は false を返します。
(>)(n)
self が n より大きければ true、それ以外は false を返します。
(<=)(n)
self が n より小さいか等しければ true、それ以外は false を返します。
(>=)(n)
self が n より大きいか等しければ true、それ以外は false を返します。
(+)(n)
self と n の和を返します。
(-)(n)
self と n の差を返します。
(*)(n)
self と n の積を返します。
(/)(n)
self と n の商を返します。
(**)(n)
self の n 乗を返します。
(-@)()
self の正負を逆転させた数値を返します。

その他のメソッドの定義は、初期化ファイル群の中の int.cy にあります。
