文字列

"abc" や "John Smith" など、文字列を表すオブジェクトです。 評価すると、自身を返します。

ペアレント

String

メソッド

(<)(str)
self が str より小さければ true、それ以外は false を返します。
(>)(str)
self が str より大きければ true、それ以外は false を返します。
(<=)(str)
self が str より小さいか等しければ true、それ以外は false を返します。
(>=)(str)
self が str より大きいか等しければ true、それ以外は false を返します。
(+)(str)
self と str を結合した文字列を返します。
to_i()
self を数値化した値を返します。
intern()
self を intern したシンボルを返します。
istring()
self を読み取れる読み取り専用のストリームを返します。
ostring()
self を書き込んである書き込み専用のストリームを返します。
